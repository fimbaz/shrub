#shrub 
###shrub is a TUI pet plant simulator currently featuring
- water
- wind
- dirt
- plants
- pollen
- seeds
- genes
- reproduction 
- death
- decay

Controls:
hjkl <- move cursor
HJKL <- jump cursor
s    <- plant seed @ cursor
d    <- cut plant
0..6 <- resource view

Interesting things:
-Male flowers are pink
-female flowers are purple
-pollen is '.'
-new seeds are usually '&'


Known limitations:
-no saving
-friggin C
-limited UI
-may segfault with certain libc
